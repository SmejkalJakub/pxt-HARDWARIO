# pxt-hardwario

Extension for micro:bit. With this extension you can control sensors and tags from the HARDWARIO IoT Kit.

Supported Modules and Tags(Modules with "#" are still in developement):
  - Battery Module
  - Mini Battery Module
  - PIR Module#
  - CO2 Module
  - Climate Module
  - LCD Module#
  - GPS Module#
  - Infra Grid Module#
  - Relay Module
  - Tag Module (only right side will work)
  - Temperature Tag
  - Humidity Tag
  - Barometer Tag
  - Lux Meter Tag
  - VOC Tag
  - VOC-LP Tag#
  - NFC Tag#
